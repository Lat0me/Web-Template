# Front office - Template 

## Install

In `front`

```bash 
# with npm 
npm install 

# with Yarn
yarn install 
```

### Parcel Commands

```bash 
# with npm 
npm run dev
npm run dev-watch-update
npm run build

# with Yarn
yarn dev-watch-update
yarn build
```

## Dependency 

- React-Bootstrap 
- leaflet, React-Leaflet
- Moment, React-Moment
